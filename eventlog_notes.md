# Windows Event Log Analysis

## Objective
Monitor Windows Security logs using Event Viewer.

## Event IDs Reviewed

### Event ID 4624
Successful logon.

### Event ID 4625
Failed logon attempt.

## Observations

- Security logs store authentication events.
- Failed login attempts are useful for detecting brute-force attacks.
- Event Viewer helps SOC analysts investigate incidents.

## Conclusion

This project demonstrates basic Windows log monitoring and security event analysis.
