# 🛡️ SOC Home Lab

## Overview

This project demonstrates basic Security Operations Center (SOC) monitoring using Windows Event Viewer.

## Objectives

- Monitor Windows Security Logs
- Analyze successful and failed login events
- Identify Event IDs
- Document security observations

## Tools Used

- Oracle VirtualBox
- Windows 11
- Event Viewer

## Key Event IDs

| Event ID | Description |
|----------|-------------|
| 4624 | Successful Logon |
| 4625 | Failed Logon |
| 4634 | Logoff |
